# Paper Review LLM

This repository stores advanced LLM-generated paper reviews. It utilizes multi-angle system prompts (e.g., positive, negative, and cautious/RAID-focused) to evaluate the security, systems architecture, and ethical dimensions of academic publications.

## Contents
